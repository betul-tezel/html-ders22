HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan bir işaretleme dilidir. Temel yapısı şu şekildedir: html Kodu kopyala Sayfa Başlığı

Başlık

Bu bir paragraftır.

Açıklamalar: : HTML5 belgesi olduğunu belirtir. : HTML belgesinin başlangıcı. : Meta bilgiler, stil ve başlık gibi içeriklerin bulunduğu bölüm. : Karakter seti ve görünüm gibi bilgileri tanımlar. : Tarayıcı sekmesinde görünen sayfa başlığı. <body>: Sayfanın görünen içeriği. <h1>: Başlık etiketidir (1. seviye). <p>: Paragraf etiketidir. Bu yapı, HTML belgesinin temelini oluşturur. İşte temel HTML komutları ve açıklamaları: 1. Başlık Etiketleri <h1> - <h6>: Başlık etiketleri, 1'den 6'ya kadar hiyerarşik başlık seviyeleri. html Kodu kopyala <h1>Başlık 1</h1> <h2>Başlık 2</h2> 2. Paragraf <p>: Paragraf oluşturur. html Kodu kopyala <p>Bu bir paragraftır.</p> 3. Bağlantı <a href="URL">: Başka bir sayfaya bağlantı oluşturur. html Kodu kopyala <a href="https://www.example.com">Example Sitesi</a> 4. Görüntü <img src="URL" alt="Açıklama">: Resim ekler. html Kodu kopyala <img src="resim.jpg" alt="Açıklayıcı metin"> 5. Liste Sıralı Liste: <ol> ve <li> etiketleri kullanılır. html Kodu kopyala <ol> <li>Birinci öğe</li> <li>İkinci öğe</li> </ol> Sırasız Liste: <ul> ve <li> etiketleri kullanılır. html Kodu kopyala <ul> <li>Öğe 1</li> <li>Öğe 2</li> </ul> 6. Tablo <table>, <tr>, <td>, <th>: Tablo oluşturur. html Kodu kopyala <table> <tr> <th>Başlık 1</th> <th>Başlık 2</th> </tr> <tr> <td>Hücre 1</td> <td>Hücre 2</td> </tr> </table> 7. Form <form>, <input>, <label>, <textarea>, <button>: Kullanıcıdan veri almak için form oluşturur. html Kodu kopyala <form> <label for="name">İsim:</label> <input type="text" id="name" name="name"> <button type="submit">Gönder</button> </form> 8. Yapısal Etiketler <div>: Bölüm oluşturur. <span>: Satır içi bir bölüm oluşturur. 9. Yorum <!-- Bu bir yorumdur -->: HTML belgesinde yorum yazmak için kullanılır. Bu temel komutlar, HTML ile sayfa oluştururken sıkça kullanılır.
