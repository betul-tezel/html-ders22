1. Element Seçicileri Tanım: Belirli bir HTML etiketini (elementi) seçer. Örnek: css Kodu kopyala p { color: blue; /* Tüm

etiketlerinin metnini mavi yapar */ } 2. Sınıf Seçicileri Tanım: . ile başlayan, belirli bir sınıfa sahip öğeleri seçer. Bir sınıf, birden fazla öğeye uygulanabilir. Örnek: css Kodu kopyala .button { background-color: green; /* .button sınıfına sahip tüm öğelerin arka planını yeşil yapar */ } 3. ID Seçicileri Tanım: # ile başlayan, benzersiz bir ID’ye sahip öğeleri seçer. Her ID yalnızca bir öğe için kullanılmalıdır. Örnek: css Kodu kopyala #header { font-size: 24px; /* #header ID’sine sahip öğenin font boyutunu 24px yapar */ } 4. Öznitelik Seçicileri Tanım: Belirli bir özniteliğe sahip öğeleri seçer. Örnek: css Kodu kopyala input[type="text"] { border: 1px solid #ccc; /* Metin girişi alanlarının kenarlığını ayarlar */ } 5. Hiyerarşik Seçiciler Tanım: Öğelerin yapısal ilişkilerini kullanarak seçim yapar. Belirli bir öğenin altındaki veya içindeki diğer öğeleri hedefler. Örnek: css Kodu kopyala ul li { list-style-type: square; /* Tüm

    içindeki
  • öğelerinin işaretini kare yapar */ } 6. Grup Seçicileri Tanım: Birden fazla seçiciyi bir arada kullanarak aynı stil kurallarını uygulamak için kullanılır. Örnek: css Kodu kopyala h1, h2, h3 { color: red; /* Tüm h1, h2 ve h3 etiketlerinin rengini kırmızı yapar */ } 7. Pseudo-seçiciler Tanım: Belirli bir durumu (örneğin, fare ile üzerine gelme) hedefler. Örnek: css Kodu kopyala a:hover { text-decoration: underline; /* Fare ile üzerine gelindiğinde bağlantının altını çizer */ } Özet CSS seçicileri, bir web sayfasındaki öğeleri hedeflemek ve onlara stil vermek için kullanılır. Temel seçici türlerini bilmek, sayfanızın görünümünü ve düzenini kontrol etmenizi sağlar. Seçicilerin doğru kullanımı, daha etkili ve düzenli stil sayfaları oluşturmanıza yardımcı olur.
